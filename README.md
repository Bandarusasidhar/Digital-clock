# ⏰ Digital Clock

A responsive Digital Clock built using *HTML, **CSS, and **JavaScript*. The application displays the current time, AM/PM format, current day, and current date in real time.

## 📸 Screenshot

![Digital Clock](Screenshot.png)

## ✨ Features

- Real-time digital clock
- 12-hour time format (AM/PM)
- Live seconds update
- Current day display
- Current date display
- Responsive and clean user interface
- Gradient background with modern styling

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox)
- JavaScript (ES6)
- DOM Manipulation
- Date Object
- Functions
- Conditional Statements
- setInterval()
